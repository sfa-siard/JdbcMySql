JdbcMySql - Users' Manual - Introduction

Introduction

JdbcMySql was developed by Enter AG for the Swiss Federal Archives as part of the SIARD (Software-Independent Archival of Relational Databases) project.

It serves the purpose of providing standardized access to relational databases based on SQL:2008. Proprietary deviations from the standard can be hidden behind this façade.

This document is the users' manual of JdbcMySql.
